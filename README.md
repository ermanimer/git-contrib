# git-contrib

Creates Simple Contribution Report For Git

# Installation 

```bash
go install github.com/ermanimer/git-contrib@latest
```

# Usage

Run following command in a git repository.

```bash
git-contrib
```

# Sample Output

```bash
+---------------------+------------+------------+
| author              | line count | percentage |
+---------------------+------------+------------+
| ermanimer@gmail.com |        268 |     100.00 |
+---------------------+------------+------------+
```
